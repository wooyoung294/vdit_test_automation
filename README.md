## VDIT 테스트 자동화
 - VDIT 앱의 아래 3가지 시나리오 항목을 Appium을 이용해 BDD 방식으로 테스트 자동화를 구현하고 테스트 결과를 Allure-report를 이용해 출력한 프로젝트 입니다.  

### TestReport : vdit.wooyoung.site
 - 각 케이스의 Teardown에 첨부된 영상은 맥북에서 실행해주시면 감사하겠습니다.

### 테스트 환경
 - iOS Version : 26.0.1
 - VDIT Version : 3.5.3
   
### 테스트 시나리오
 - 언어변경 KR -> EN
 - 캔버스 비율 변경 원본 -> 1:1
 - 텍스트 삭제

### 사용 언어 및 기술
 - Python
 - Appium
 - Pytest
 - Pytest-bdd
 - Allure-report
